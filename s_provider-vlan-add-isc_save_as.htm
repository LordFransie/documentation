 ?.1 Add Provider Network
TO PROVIDE FEEDBACK ON THIS HELP PAGE ⇒ CLICK HERE
Adding a Provider Network enables you to provision an existing data center network to the cloud.
Prerequisites
 • Minimum required privileges: Infrastructure Administrator
 • Cloud Networking is configured.
Adding a Provider Network
 1. From the main menu, select Provider Networks.
 2. Click + Add Network.
 3. Enter data for this network. See [ ] .
 4. On the Add Provider Network screen, enter a Name and VLAN ID for this network.
 5. If you do not want this network to be shared by other components, such as virtual machines and hypervisors, clear the Shared check box.
 6. If you do not want this network to forward packets, clear the Admin State Up check box.
 7. Optional: To add a subnet to this network, do one of the following.
 • To add a subnet to a new network:
 1. Click Add subnet.
 2. Enter data for this subnet. See [ ] .
 3. On the Add Subnet screen, enter an IPv4 address in CIDR format to specify the IP address range available to this network.
 4.  If the IP addresses listed for Allocation Pools or Gateway IP are not correct, change the default values.
 5. If the network already has a DHCP server, clear the Enable DHCP check box.
 6. Click OK.
 7. Verify that the new subnet is displayed on the Add Provider Network screen. To sort by CIDR, select the CIDR column heading.
 • To add a subnet to an existing network:
 1. On the Provider Networks overview screen, select the row of the network to which you want to add a subnet.
 2. Select ActionsEdit.
Alternatively, hover over the details of the selected network to display the Edit icon, and then click the Edit icon.
 3. Enter data for this subnet. See [ ] .
 4. On the Add Subnet screen, enter an IPv4 address in CIDR format to specify the IP address range available to this network.
 5.  If the IP addresses listed for Allocation Pools or Gateway IP are not correct, change the default values.
 6. If the network already has a DHCP server, clear the Enable DHCP check box.
 7. Click OK.
 8. Verify that the network update was successful by reviewing the fields on the Edit Provider Networks screen. To sort by CIDR, select the CIDR column heading.
 9. Finish adding the network.
 • To add only this network, click Add.
The new network displays on the overview screen.
 • To add more than one network:
 1. Click Add+ to complete the addition process for the first network and reset the form.
The Name and VLAN ID fields are cleared, but the other options remain checked for future use. 
 2. Enter a unique Name and VLAN ID for the network.
 3. Update other options if needed.
 4. Repeat steps 1, 2, and 3 until you are finished adding additional networks, then click Cancel to dismiss the Add Provider Network screen. 
 5. Verify that each new network is displayed on the Provider Networks screen. To sort by network name, select the Name column heading.
